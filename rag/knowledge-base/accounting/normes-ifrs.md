# Normes IFRS - Référence Rapide

## IFRS 9 - Instruments Financiers

### Classification
- **Amortized Cost**: Dettes, créances à revenus fixes
- **FVOCI** (Fair Value Through OCI): Actifs à revenus variables
- **FVTPL** (Fair Value Through P&L): Trading, dérivés

### Dépréciation (Expected Credit Loss)
```
Stage 1: 12-month ECL (performing)
Stage 2: Lifetime ECL (deteriorated)
Stage 3: Lifetime ECL (credit-impaired)
```

## IFRS 15 - Revenus Contrats Clients

### 5 Étapes
1. Identification contrat
2. Identification obligations
3. Détermination prix transaction
4. Allocation prix
5. Reconnaissance revenu

## IFRS 16 - Location

### Locataire
- Toutes locations > 12 mois au bilan
- Actif: Droit d'utilisation
- Passif: Dette de location

### Ratios impactés
- **Endettement**: +++ (dettes augmentent)
- **ROA**: -- (actifs augmentent)
- **EBITDA**: ++ (charges reclassées)
